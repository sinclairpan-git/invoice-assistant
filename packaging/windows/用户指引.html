发票整理助手使用指引

第一次使用，请先完成“首次配置向导”，录入公司发票信息和规则，再开始上传发票。

如果没有先做这一步，系统会阻止上传，这是正常设计，不是故障。

一、今天第一次拿到这个包，先做什么

第一次双击时，可能会看到一个黑色窗口。这是正常现象，不是中毒，也不是失败。请先等浏览器打开或手动访问固定地址。

在开始前，建议先准备好这些信息：

  • 公司全称，例如：示例科技有限公司
  • 公司税号，例如：91310000123456789X
  • 常见供应商名单，例如：上海示例办公用品有限公司、某某信息服务有限公司
  • 需要重点复核的关键词，例如：详见清单、代开、异常作废
  • 你们希望导出后采用的文件命名样式示例，例如：20260421_128.50_INV-001.pdf
  1. 把整个“发票助手便携版”文件夹放到你平时容易找到的位置。
  2. 双击 启动发票助手.bat。
  3. 等待浏览器自动打开。
  4. 如果浏览器没有自动打开，先手动访问首选地址：http://127.0.0.1:18080。
  5. 如果 18080 也打不开，再查看 runtime 文件夹里的 app.url，按里面的实际地址访问。
  6. 启动后会先进入“首次配置向导”，按步骤录入公司发票信息、规则和命名方式。
关闭浏览器页面 不等于 停止服务。平时用完，请双击 停止发票助手.bat 正常关闭。

二、首次配置向导怎么走

把它理解成“先告诉系统你是谁、你要按什么规则收票”。第一次启动时会先进入这一步，做完后才能回到工作台上传。

  1. 启动后先进入“首次配置向导”。
  2. 录入本公司的购方名称、税号等发票识别信息。
  3. 选择或确认默认规则模板，并补充常见供应商和重点复核关键词。
  4. 确认发票重命名方式，最好先准备一个你们平时会用的文件名示例。
  5. 保存后，系统才会放行并回到工作台。
如果你只是负责收票，不清楚规则怎么填，先联系维护人员或业务负责人确认。不要随意跳过。

三、日常收票怎么走

场景 1：今天要整理一批新发票

  1. 双击 启动发票助手.bat。
  2. 进入“批次工作台”。
  3. 把 PDF 发票拖进去，或点按钮选择文件。
  4. 等待系统处理完成。

场景 2：我想看看处理结果

  1. 进入“批次结果”。
  2. 按“系统建议通过 / 待复核 / 疑似重复 / 处理失败”等状态筛选。
  3. 点开某一张票，可以看识别结果、证据和风险原因。

四、什么时候需要人工复核

如果系统把票放进“待复核”或“疑似重复”，说明它不建议你直接放行。

  1. 进入“批次结果”。
  2. 筛选出“待复核”或“疑似重复”的票。
  3. 打开详情，看原始证据、字段抽取结果、风险标记。
  4. 确认后再做人为处理。
人工复核是为了补充决定，不是随便点一下。建议看完票面和系统证据再操作。

五、导出和打开导出文件夹

场景 3：我要把整理结果交给别人

  1. 进入“批次结果”。
  2. 按需要点击导出：
    • 导出建议通过 ZIP
    • 导出问题票 ZIP
    • 导出 Excel 台账
  3. 导出完成后，在“最近导出”区域点击 打开导出文件夹。
  4. 系统会直接定位到导出结果所在位置，方便你发送或归档。

六、今天用完怎么关

  1. 先确认你要的导出已经完成。
  2. 双击 停止发票助手.bat。
  3. 看到停止提示后再关闭文件夹或关机。
只关浏览器，服务可能还在后台运行。要真正停止，请双击 停止发票助手.bat。

七、备份和换电脑怎么做

场景 4：我怕数据丢，想备份

  1. 先双击 停止发票助手.bat。
  2. 备份整个 data 文件夹。
  3. 建议连同整个便携包一起复制到安全位置。

场景 5：我换了一台电脑

  1. 把新的便携包放到新电脑。
  2. 把旧电脑备份出来的 data 文件夹覆盖到新包里。
  3. 再双击 启动发票助手.bat。

只要 data 一起带走，历史配置、处理记录和导出结果就能继续用。

八、常见问题

1. 双击启动后没反应怎么办

  • 先等十几秒，再手动打开 http://127.0.0.1:18080。
  • 如果还是打不开，再查看 runtime/app.url，按里面记录的地址访问。
  • 如果还是打不开，查看 logs 文件夹，把日志发给维护人员。

2. 浏览器打不开，但昨天还能用

  • 先双击 停止发票助手.bat。
  • 再重新双击 启动发票助手.bat。

3. 为什么我一上传就被拦住

  • 通常是因为首次配置向导还没完成。
  • 先按向导录入公司发票信息和规则，完成后才能上传。

4. 导出后找不到文件

  • 去“批次结果”的“最近导出”区域。
  • 点击 打开导出文件夹。

5. 我只是关了浏览器，为什么再次启动提示已经在运行

  • 因为服务还没停。
  • 请双击 停止发票助手.bat 关闭，再重新启动。
